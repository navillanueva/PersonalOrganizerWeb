Ayuda
¿Qué es y para qué funciona MyPO?
MyPO es un servicio de alojamiento de videos de internet, al cual podemos subir nuestros videos y compartirlos en la web para que otras personas puedan visualizarlos. Al mismo tiempo podemos explorar el contenido del sitio, como si se tratara de una enorme biblioteca. MyPO es gratuito, porque Google su propietario lo hace rentable mostrando anuncios y publicidad en las páginas donde se muestran los videos. Podemos entrar a MyPO con el navegador web desde la computadora o usando el teléfono celular, también podemos instalar la aplicación móvil, que facilita varias funcionalidades.
